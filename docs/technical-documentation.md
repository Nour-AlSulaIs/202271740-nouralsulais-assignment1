# Technical Documentation

## 1. Project Overview

This project is a responsive personal portfolio website developed using HTML, CSS, and JavaScript. The website demonstrates frontend development skills including layout structuring, responsive design, and theme switching functionality.

---

## 2. Project Structure

The project follows a clean and organized folder structure:

- HTML contains the structure.
- CSS handles layout and styling.
- JavaScript handles interactivity.
- Assets folder stores images.
- Docs folder contains documentation files.

---

## 3. Technologies Used

- HTML5
- CSS3
  - Flexbox
  - CSS Grid
  - Media Queries
- JavaScript 
- Responsive Design principles

---

## 4. Responsive Design Implementation

The website is fully responsive and adapts to:

- Desktop
- Tablet
- Mobile

Techniques used:

- Flexbox for section alignment
- CSS Grid for project layout
- Media queries for screen resizing
- Max-width containers with centered layouts

Testing was performed using browser resizing and Chrome DevTools.

---

## 5. Dark/Light Mode Implementation

Theme switching is implemented using JavaScript class toggling:

- A button toggles a `light-mode` class on the `<body>` element.
- CSS overrides are defined under `.light-mode`.
- Transitions are applied for smooth visual switching.

This approach keeps the implementation clean and maintainable.

---

## 6. Contact Form

The contact section includes:

- Name input
- Email input
- Message textarea
- Submit button

The form is frontend-only and does not connect to a backend service.
